# **App Name**: EcoTrust Exchange

## Core Features:

- Unified Sign-Up: Register a new user account with email and wallet address, generating a secure password sent to the user's email.
- Unified Login: Authenticate users by matching their wallet address and provided password.
- Carbon Credit Marketplace: Browse a list of available carbon credit projects with associated details. Authenticated user only.
- Admin Verification Queue: Admins or Verifiers can review projects awaiting verification from the GET API.
- Email Simulator: For development purposes, simulates the sending of emails by logging generated passwords to the console.

## Style Guidelines:

- Primary green (#10B981) for "buy", "approve", and general eco-related elements, suggesting growth and sustainability.
- Primary blue (#3B82F6) for "login", "trust", and tech elements, evoking feelings of security and technological advancement.
- Accent yellow (#F59E0B) to highlight warnings or important information, drawing user attention.
- Danger red (#EF4444) for "sell", "reject", and error messages, communicating urgency and potential issues.
- Neutral background (#F9FAFB) for the page to provide a clean and unobtrusive backdrop.
- Neutral card background (#FFFFFF) for cards and modals to create a visually distinct content area.
- Primary text color (#1F2937) for headings and other dark text elements, ensuring readability.
- Secondary text color (#6B7280) for body text and subheadings, providing a softer contrast against the background.
- Font: 'Inter', a sans-serif font for all text, providing a clean and modern aesthetic. Note: currently only Google Fonts are supported.
- Fully dynamic and responsive design, optimized for mobile-first development.
- All elements (buttons, inputs, cards) will have rounded corners (rounded-lg) and subtle shadows (shadow-md).
